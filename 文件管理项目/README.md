# 文件系统管理项目

本项目使用 Python + PyQt 实现一个简单文件系统模拟器。程序在内存中模拟磁盘空间，使用 FAT 显式链接管理文件物理块，使用位图管理空闲块，并支持多级目录、文件读写、删除、保存和加载。

## 功能

- 格式化模拟磁盘
- 创建、删除子目录
- 显示目录树和当前目录内容
- 更改当前目录、返回上级目录
- 创建、打开、关闭、写入、读取、删除文件
- 资源管理器式界面：菜单栏、路径栏、目录树和文件详细列表
- 查看 FAT 表和位图
- 自动保存和手动保存文件系统状态

## 文件结构

```text
main.py          程序入口
ui_main.py       PyQt 主界面
dialogs.py       输入和信息弹窗
filesystem.py    文件系统核心逻辑
models.py        文件和目录数据结构
storage.py       保存/加载 filesystem.dat
test_core.py     核心逻辑自检脚本
```

## 文件系统设计

- 磁盘块数量：128 块
- 每块大小：64 字节
- 总容量：8192 字节
- 文件分配方式：FAT 显式链接
- 空闲空间管理：位图
- 目录结构：多级目录树

FAT 表约定：

```text
-2 表示空闲块
-1 表示文件结束
其他数字表示下一块编号
```

## 运行

进入 conda 环境后，在当前目录执行：

```bash
python main.py
```

## 打包

使用 PyInstaller 打包：

```bash
pyinstaller -F -w main.py -n FileSystemManager
```

生成的可执行程序位于：

```text
dist/FileSystemManager.exe
```

## 演示建议

1. 启动程序并格式化文件系统。
2. 创建 `docs`、`images` 等目录。
3. 双击进入目录，观察顶部路径栏变化。
4. 新建文本文件，双击打开编辑窗口并写入较长内容。
5. 查看 FAT 表与位图，说明文件块之间的链接关系。
6. 重命名文件或文件夹，验证目录树和文件列表同步更新。
7. 删除文件，查看 FAT 表与位图确认磁盘块释放。
8. 保存后关闭程序，再次启动，验证目录和文件仍然存在。
